Sicherheit in Pecunia

Pecunia kann Ihre Konto- und Umsatzdaten auf Wunsch verschlüsseln. Die Verschlüsselung erfolgt mit dem AES-256-Verfahren. Wählen Sie bei den Pecunia-Einstellungen den Reiter Sicherheit. Dort können Sie angeben, ob die Daten verschlüsselt werden sollen und ob die Datendatei in einem anderen Ordner (z.B. USB-Stick) liegen soll. Wenn Sie die Daten verschlüsseln möchten müssen Sie im nächsten Schritt ein Passwort angeben. Sie können dieses Passwort im Schlüsselbund sichern. Wenn Sie das Passwort nicht im Schlüsselbund sichern muss es bei jedem Start von Pecunia angegeben werden.
Auf diese Weise können Sie also gleichzeitig einen Passwortschutz von Pecunia bei Programmstart erreichen.
