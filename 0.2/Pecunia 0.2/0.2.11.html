Release Information Pecunia 0.2.11

In Pecunia 0.2.11 wurden gegenüber der Version 0.2.9 und 0.2.10 folgende Fehler bzw. Probleme behoben:
  • Doppelte Umsätze wenn die Bank zusätzliche Leerzeichen schickt
  • Sperrprobleme bei Version 0.2.10
  • Springen in das Jahr 2100 oder 2099 falls die Bank Umsätze von 1999 oder früher schickt
  • Timeout bei Netzwerk- oder USB-Last
Außerdem gibt es eine Möglichkeit zum Importieren von Umsätzen aus anderen Programmen bzw. Dateien.

Wichtig!!

Wenn Sie vorher 0.2.9 verwendet haben: es gibt bei der AqBanking-Migration auf Version 5.0 noch einen Bug, der alle Konten auf Sammelüberweisungen umstellt. Bitte ändern Sie alle Konten für die Sie keine Sammelüberweisung wünschen und löschen Sie das Flag "Sammelüberweisung".
