Überweisungen durchführen

Sie können in Pecunia Standardüberweisungen (Inlandsüberweisungen), Terminierte Überweisungen und EU-Überweisungen durchführen. Um eine Überweisung durchzuführen wählen sie zuerst das Konto aus, das belastet werden soll und wählen dann eine der folgenden Überweisungsoptionen:
Anlegen einer Inlandsüberweisung.
Anlegen einer Terminierten Überweisung. Hier müssen Sie ein Datum eingeben, zu dem die Überweisung ausgeführt werden soll.
Anlegen einer EU-Überweisung. Hier müssen Sie ein IBAN und BIC des Empfängers spezifizieren.
Im anschließenden Überweisungsdialog geben Sie wie gewohnt die Überweisungsdaten an. Wählen Sie Nächste Überweisung wenn Sie für das gleiche Konto eine weitere Überweisung anlegen möchten und Fertig wenn Sie keine weiteren Überweisungen mehr anlegen möchten.
Überweisungen können als Vorlage markiert und später über die Vorlagenauswahl kopiert werden. Dies ist dringend zu empfehlen wenn regelmäßig Überweisungen an gleiche Empfänger durchgeführt werden, da es das Risiko der Fehleingabe minimiert. Die Bank haftet nämlich nicht für Überweisungen, die versehentlich an falsche Konten geschickt wurden.

Im Überweisungsdialog können Sie die Überweisung als Vorlage (Sichern als Vorlage unter) speichern und sie dann später unter dem gewählten Namen wiederfinden.
Die Überweisungen werden nicht direkt an die Bank gesendet sondern zunächst in der Überweisungsliste gesammelt. Dort können sie dann geprüft und ggfs. gesammelt an die Bank übermittelt werden. Wenn Sie Pecunia schließen und es befinden sich noch nicht gesendete Überweisungen in der Überweisungsliste, wird eine entsprechende Informationsmeldung ausgegeben.

Sie können für jedes Konto über Konto/ändern... bestimmen, ob die Überweisungen gebündelt als Sammelüberweisung oder einzeln an die Bank gesendet werden. Beide Verfahren haben Vor- und Nachteile. Bei der Sammelüberweisung können Sie viele Überweisungen mit einer einzigen TAN versenden. Die Bank weist dann aber auch nur einen Sammelumsatz aus, so dass die Informationen über die einzelnen Buchungen verloren gehen.

Überweisungsvorlagen verwalten

Sie können über das Menü Bank/Überweisungsvorlagen... Ihre Überweisungsvorlagen bearbeiten. Jede Überweisung hat einen Namen bzw. eine Beschreibung, damit sie später bei der Überweisung schnell gefunden werden kann. Sie können über die Buttonleiste unter der Liste Vorlagen anlegen, löschen und ändern.
