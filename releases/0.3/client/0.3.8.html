Release Information Pecunia 0.3.8

In Pecunia 0.3.8 wurden gegenüber der Version 0.3.7 folgende Fehler bzw. Probleme behoben:
  • Probleme Umsatzdupletten
  • Kontoabfrage bei manuellen Konten
  • Sortierung der Umsätze beim Export
  • Entfernen von PINs aus dem Schlüsselbund
  • Doppelte Toolbar-Icons
Aufgrund zahlreicher Probleme mit der neuen AqBanking-Version in Pecunia 0.3.7 enthält Pecunia 0.3.8 nun wieder die gleiche Version wie Pecunia 0.3.6.
Bei Problemen mit Version 0.3.8 bitte wieder die 0.3.7 einspielen oder die demnächst erscheinende Version 0.4 verwenden.
