Raw Ideas

Allgemeines
-----------
- Start von Pecunia mittels Alt+Klick zur Auswahl einer Datenbank (zusätzlich zum Kommandozeilenparameter).
- Unterstützung für Paypal.
- Unterstützung für Chipkarten
- Navigationshistorie + back/forward buttons für alle Seiten
- Kontenmappen, also eine Möglichkeit für verschiedene Eigentümer die Konten anzuzeigen (und Kategorien). Vielleicht täte es auch ein Filter.
- Eine Startseite:
"ich dachte da an verschiedene Dinge, z.B ein Modul für Konten, so das man einen Direkteinstieg hat auf Konten, Kreditkarten. Etwa mit einem Rechtsklick eine Überweisung direkt von einem Konto anstoßen kann. Oder eine Statistik der Ausgaben-Einnahmen. Fällige, noch nicht getätigte Überweisungen. Aktienkurse, Devisenmärkte eventuell einen integrierten Taschenrechner, Adressbuch, Abruf der einzelnen Konten. Eine individuell anpassbare Startseite ist sehr bequem für den Benutzer."
- OCR zur automatischen Übernahme von vorgedruckten Zahlungsanweisungen.
- Pecunia nach einer Zeitspanne automatisch sperren.
- Einstellbarer Starttag in einenm Zeitbereich (statt immer z.B. vom ersten zum letzten im Monat), z.B. über ein Setting.

Kategorie + Kontenanalyse
-------------------------
- (Graph) anpassbar durch Nutzer
- (Graph) Export zu png, pdf
- (Graph) Anzeige von min, max, average
- (Graph) Trendlinie (zumindest für Konten)
- (Graph) mehrere Kategorien zusammen
- (Graph) Auch für Kontenknoten alle Kindknoten in die Auswertung einbeziehen (wie schon für Kategorien).
- statt mehrere Kategorien in einem Zeitraum vergleiche eine Kategorie in mehreren Zeiträumen
- Umsatzsteuerschlüssel für jede Kategorie.
- Nutzerdefiniertes Icon für eine Kategorie.
- Vordefinierte/automatische Kategorien - anlegen (einschließlich Regeln) bei Erstabruf eines Kontos.
- Numerischer Kategorienexport (List + Einnahmen/Ausgaben).
